To use this demo, you need to:

  1. Get an authentication token from your VoiceCake account
  2. Replace the placeholder token in the script tag below
  3. Make sure the agent ID is correct

Step 1: Get Your Auth Token

You can get your authentication token by:

  • Logging into your VoiceCake dashboard
  • Going to Settings ‚Üí API Keys
  • Copying your access token

Step 2: Update the Script Tag

Replace YOUR_AUTH_TOKEN_HERE with your actual token:

<script src="voicecake-widget-loader.js" data-agent-id="111" data-auth-token="YOUR_AUTH_TOKEN_HERE" data-debug="true"> </script>

Step 3: Test the Widget

Once you've added your auth token:

  1. Click the floating microphone button in the bottom-left corner
  2. Click "Start Chat"
  3. Speak into your microphone
  4. You should hear the agent respond!

How to Embed the Widget

Add this script tag to your website's <head> section:

Configuration Options

data-agent-id="YOUR_AGENT_ID" // Required: Your VoiceCake agent ID data-api-url="https://..." // Optional: API base URL data-ws-url="wss://..." // Optional: WebSocket base URL data-debug="true|false" // Optional: Enable debug logging data-position="bottom-left|bottom-right" // Optional: Widget position data-theme="default|dark|light" // Optional: Widget theme
